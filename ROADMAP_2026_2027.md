# Product Roadmap 2026–2027 — IDA Dataset Factory

**Architecture:** Frozen at v2.0  
**North star:** Continuously higher **coverage · quality · freshness · automation · export**  

Related: [BACKLOG.md](./BACKLOG.md) · [RELEASE_PLAN.md](./RELEASE_PLAN.md) · [MILESTONES.md](./MILESTONES.md) · [KPI.md](./KPI.md)

---

## Next 30 days (Foundation production)

**Theme:** Make the factory produce measurable growth every week.

| Focus | Epic items | Outcome |
|-------|------------|---------|
| Source health | E1-01, E1-02, E1-04, E1-13, E1-14 | BPS, World Bank, OJK usable + health status |
| Industry growth | E2-01, E7-01, E7-11 | Industry library rows & completeness ↑ |
| Extraction gates | E3-01, E3-02, E3-03, E3-06, E3-08 | No placeholders; duplicates blocked |
| QA in CI | E8-01, E8-02, E8-03, E8-06 | `quality.yml` fails on bad data |
| Export harden | E5-01, E5-03, E5-07 | Versioned JSONL + OpenAI packages |
| Monitoring | E6-01–E6-04, E6-09 | Factory KPIs accurate on dashboard |

**KPI targets (30 days)**

- Industry rows ≥ 8 (from 4) with 100% schema completeness on new rows  
- Active healthy sources ≥ 6  
- Average confidence ≥ 0.85 on new publishes  
- Zero active placeholder URLs  
- ≥ 1 automated export package per successful learn cycle

## Next 90 days (Scale the factory)

**Theme:** Multi-library expansion + mission automation.

| Focus | Epic items | Outcome |
|-------|------------|---------|
| Sources | E1-03, E1-05, E1-07, E1-09–E1-12, E1-15 | Source diversity ↑ |
| Datasets | E2-02, E2-05, E2-06, E2-08, E2-10, E2-16 | Company, pain, solution, case, regulation, competitor growth |
| Extraction | E3-04, E3-05, E3-07, E3-09, E3-10 | Entity match + calibration + fixtures |
| Quality UX | E4-01–E4-06, E4-08 | Full quality metric suite |
| Export | E5-02, E5-04, E5-08, E5-09 | Parquet + HF package + multi-dataset |
| Missions | E7-02–E7-04, E7-08–E7-10, E7-13 | Scheduled missions + retry + success rate |
| QA | E8-04, E8-05, E8-07, E8-08 | Duplicate/coverage gates; pre-export QA |
| Monitoring | E6-05–E6-08 | History + export status + targets |

**KPI targets (90 days)**

- Industry rows ≥ 25 (catalog target ladder)  
- ≥ 3 libraries with non-zero growth beyond industry  
- Duplicate rate ≤ 2% on new batches  
- Freshness ≥ 80% of rows updated in 90 days (active libraries)  
- Mission success rate ≥ 85%  
- Parquet + HF + OpenAI exports available for industry

## Next 6 months (Corpus depth)

**Theme:** Domain depth + training-package maturity.

| Focus | Outcomes |
|-------|----------|
| Dataset classes | Buyer persona, KPI, risk, trend, decision-maker libraries (E2-09–E2-15) |
| Domains | First real rows in finance / legal / operations stubs (E2-17) |
| Export formats | ShareGPT + Alpaca (E5-05, E5-06) |
| Automation | Full mission library operational (E7) |
| Quality | Trend views + CI thresholds (E4-07, E4-09) |
| Optional | HF Hub publish opt-in (E5-10) |

**KPI targets (6 months)**

- ≥ 10 product dataset classes with verified rows  
- Source diversity ≥ 12 active sources  
- Export matrix: CSV, JSON, JSONL, Parquet, OpenAI, HF, ShareGPT, Alpaca  
- Coverage progress (industry catalog) ≥ 50% of defined target  
- Quality score stable ≥ 80 under growth

## Next 12 months (Factory maturity 2027)

**Theme:** Reliable, scheduled, multi-domain knowledge factory.

| Focus | Outcomes |
|-------|----------|
| Continuous production | Weekly measurable row growth without manual firefighting |
| Multi-domain corpus | BD + ≥2 additional domains with production missions |
| Training readiness | Versioned release packages each month |
| Ops excellence | Health, retry, QA gates default-on |
| Product discipline | Zero architecture redesign sprints; backlog-only delivery |

**KPI targets (12 months)**

- Monthly release of versioned training package  
- Industry catalog target ≥ 80%  
- Mission success rate ≥ 90%  
- Average confidence ≥ 0.88  
- Duplicate rate ≤ 1% on new batches  
- All five KPI families improved YoY  

---

## Explicit non-goals (entire horizon)

- Architecture redesign  
- Repository restructure  
- RAG product  
- Decision / reasoning engine  
- Chatbot / agents  
- Knowledge graph product UI

## Version line (product, not architecture)

| Version | Window | Intent |
|---------|--------|--------|
| **v2.0** | Now | Factory reset complete |
| **v2.1** | ~30–90 days | Sources + industry scale + QA + export harden |
| **v2.2** | ~3–6 months | Multi-library + Parquet/HF + mission automation |
| **v2.3** | ~6–12 months | Multi-domain + full export matrix + monthly packages |

*(Versions improve product KPIs only—pipeline shape stays v2.0.)*
